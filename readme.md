> Internet Radio

## Features

- ASCII Animations
- Livestreaming audio
- PWA

## Dependencies

- Node > 12
- ffmpeg: https://formulae.brew.sh/formula/ffmpeg
- MP3 files that have complete metadata (artist, title, bitrate, etc). Stored in `./audio`

## Development

```sh
# Install ffmpeg
brew install ffmpeg

# Install Node dependencies
yarn

# run dev server
yarn dev

# run prod server
yarn start
```
